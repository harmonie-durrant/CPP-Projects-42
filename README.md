# CPP-Projects-42
My projects in C++ that I have done with 42Perpignan

## How to run ?

go to the folder you want to test
```bash
	cd {directory of makefile}
```

build the project
```bash
	make
```

run the created file
```bash
	./{corresponding binary name}
```

clean up the files after use
```bash
	make fclean
```
